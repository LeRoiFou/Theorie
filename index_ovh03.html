Chapitre III : Comment créer une instance Public Cloud et s'y connecter


1. Créer des clés SSH

Les clés publiques et privées SSH forment une paire asymétrique utilisée pour sécuriser l'authentification lors des connexions à distance, sans mot de passe.
La clé privée reste secrète sur votre machine (client), tandis que la clé publique est partagée avec le serveur distant, qui l'utilise pour vérifier votre identité via un défi cryptographique. Cela repose sur la cryptographie à clé publique, comme RSA ou ECDSA, où la clé publique chiffre un message que seule la clé privée peut déchiffrer.

Comment utiliser PuTTY pour les connexions SSH et l'authentification

PuTTY est un logiciel client SSH open source avec une interface utilisateur graphique. Il a été développé pour Windows mais est également disponible pour d'autres systèmes d'exploitation et comprend des fonctionnalités utiles, comme la gestion des clés SSH.
Les clés publiques et privées générées doivent également être sauvegardées dans un disque dur externe.

2. Importer les clés SSH

Dans l'espace client OVHcloud, section Public Cloud > projet concerné > Paramètres > Clés SSH, ajouter la clé publique en entrant un nom et la chaîne de clé.
Les caractères composant cette clé doivent représenter une unique ligne pour être prise en compte par OVHCloud.


3. Préparer la configuration réseau

Pour un mini projet simple, optez pour le mode Public (exposition directe IPv4/IPv6 à Internet) sans réseau privé initial ; passez directement à l'étape suivante si pas de vRack nécessaire.


4. Créer l'instance

  • Sélectionnez un modèle (flavor) petit comme Discovery pour tests (1 vCPU, 2 Go RAM)
  • Choisissez une région proche (ex. GRA pour France)
  • Sélectionnez Ubuntu 22.04, ajoutez une clé SSH publique (stockée ou directe)
  • Nommez l'instance, configurez options basiques (pas de script user-data pour débuter)
  • Confirmez réseau Public
  • Choisissez facturation horaire pour flexibilité, puis créez

5. Se connecter à l'instance

Attendez l'état "Activé" dans Instances > tableau de bord. Connectez-vous via SSH : ssh ubuntu@IP_de_votre_instance (utilisateur = ubuntu pour Ubuntu)


6. Premiers pas

Définissez un mot de passe avec sudo passwd pour activer console VNC si besoin ; installez ensuite Docker pour votre projet Python
